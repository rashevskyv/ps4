## Підготовчі роботи

1. Відформатуйте USB-флешку в **exFAT**
1. У корені USB-флешки створіть папку `PS4`, а в ній папку `UPDATE`
1. Завантажте звичайний файл оновлення для простого підняття версії прошивки, або файл оновлення для рекавері, якщо ви хочете змінити жорсткий диск на приставці, або у вас вже завантажилося оновлення на останню версію ПЗ, що не піддається зламу
	* [Звичайний файл системного оновлення]{{ include.link }} для Playstation 4 версії **{{ include.version }}**
	* [Версія для рекавері]{{ include.link_recovery }} для Playstation 4 версії **{{ include.version }}**
1. Перенесіть файл оновлення в папку `PS4/UPDATE` і перейменуйте його в `PS4UPDATE.PUP`

Якщо виникає помилка **на прошитій приставці** (наприклад **CE-30778-5** або **SU-30634-6**), насамперед переконайтеся, що файл оновлення названий `PS4UPDATE.PUP`. Якщо це не допомогло — видаліть вміст папки `update` на HDD вашої приставки, або [скористайтеся пейлоадером](payloads){:target="_blank"} **update-enabler** (є за замовчуванням у складі ps4-exploit-host)
{: .notice--warning}

Якщо виникає помилка на **не прошитій** приставке, насамперед переконайтеся, що файл оновлення названий `PS4UPDATE.PUP`, спробуйте прошити з іншим USB-накопичувачем, спробуйте перекачати файл оновлення, спробуйте зробити повну ініціалізацію приставки. **Якщо у вас проблеми з приводом, то скористайтеся [цим відео](https://www.youtube.com/watch?v=jwkuIY0UoNg){:target="_blank"}**
