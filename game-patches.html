    Применение игровых патчей на лицензионные копии игр

    Разделы

    • Что понадобится
    • Инструкция
      • Часть I - Узнаем Title ID игры
      • Часть II - Скачивание патча
    Если вам помогло или просто понравилось это руководство, то мы принимаем пожертвования.
    По вопросам, касающимся взлома и руководства, обращайтесь сюда.

    Патчи можно устанавливать только для лицензионных игр!!!

    Что понадобится

    • Fake PKG Generator 1.1.3
    • pkg-merge

    Инструкция

    Часть I - Узнаем Title ID игры

    Часто Title ID пишут прямо в названии pkg-файла игры, например EP9000-CUSA02344_00-UNCHARTEDTRILOGY-A0100-V0100.pkg - CUSA02344 и есть искомое значение. Так же обращайте внимание на страницу сайта с которого качаете игру. Код диска: CUSA02344 на тапочках несёт в себе именно значение искомого Title ID. Если у вас дисковая версия игры, то код можно обнаружить на самом диске. Если же простым способом узнать его не получилось, следуйте инструкциям из этой части

    1. Извлеките содержимое архива с Fake PKG Generator в папку, путь к которой не содержит кириллицы
    2. Запустите orbis-pub-chk.exe из папки с Fake PKG Generator
    3. Перетащите .pkg файл с игрой на окно программы
    4. В окне программы появится информация об игре:
      • Content ID - содержит в себе закодированное значение региона игры, где ключевыми являются первые две бквы
        • EPхххх - EUR, европейский регион
        • UPхххх - USA, американский регион
        • JPхххх - JPN, японский регион
      • Title ID - искомое значение кода диска
      • Title Name (default) - название игры
      • Application Version - версия игры
      • Required System Software Version - требуемая версия прошивки для запуска игры

    Часть II - Скачивание патча

    1. Перейдите на сайт Playstation 4 Package Finder
    2. Введите Title ID игры для которой вы хотите скачать патч в текстовое поле и нажмите кнопку “Search”
      • Обратите внимание на строку “Minimum System Firmware”. Если её значение выше, чем 6.72, то игра не запустится на вашей приставке после установки патча!
      • Если на сайте нет патча для вашей игры, подходящей для прошивки 6.72 или ниже, поищите его в этом архиве. Обязательно проверьте совместим ли скачанный патч с прошивкой 6.72 или ниже. Сделать это можно методом из части I через orbis-pub-chk.exe.
    3. Скачайте файл формата .pkg, находящиеся в разделе Piece 0
    4. Если кроме Piece 0 есть еще и Piece 1 и так далее, то вам нужно будет скачать и их, а затем объединить все скачанные патчи в один
      1. Разархивируйте утилиту pkg-merge к скачанным частям патча
      2. Создайте папку “pkg” и перенесите в неё все патчи
      3. Запустите pkg-merge.bat
      4. Дождитесь завершения процесса. На выходе вы получите большой файл патча, который и нужно будет установить
    5. Установите .pkg-файл патча таким же способом, как и игру
