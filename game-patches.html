    Застосування ігрових патчів на ліцензійні копії ігор

    Розділи

    • Що знадобиться

    Патчі можна встановлювати тільки для ліцензійних ігор!!!

    Що знадобиться

    • Fake PKG Generator 1.1.3
    • pkg-merge

    Інструкція

    Частина I — Дізнаємося Title ID гри

    Часто Title ID пишуть прямо в назві pkg-файлу гри, наприклад EP9000-CUSA02344_00-UNCHARTEDTRILOGY-A0100-V0100.pkg — CUSA02344 і є шукане значення. Також звертайте увагу на сторінку сайту, з якого качаєте гру. Код диска: CUSA02344 на тапочках несе в собі саме значення шуканого Title ID. Якщо у вас дискова версія гри, то код можна виявити на самому диску. Якщо ж простим способом дізнатися його не вийшло, дотримуйтесь інструкцій з цієї частини

    1. Витягніть вміст архіву з Fake PKG Generator у папку, шлях до якої не містить кирилиці
    2. Запустіть orbis-pub-chk.exe з папки з Fake PKG Generator
    3. Перетягніть .pkg файл із грою на вікно програми
    4. У вікні програми з’явиться інформація про гру:
      • Content ID — містить у собі закодоване значення регіону гри, де ключовими є перші дві літери
        • EPхххх — EUR, європейський регіон
        • UPхххх — USA, американський регіон
        • JPхххх — JPN, японський регіон
      • Title ID — шукане значення коду диска
      • Title Name (default) — назва гри
      • Application Version — версія гри
      • Required System Software Version — необхідна версія прошивки для запуску гри

    Частина II — Скачування патча

    1. Перейдіть на сайт Playstation 4 Package Finder
    2. Введіть Title ID гри, для якої ви хочете завантажити патч, у текстове поле та натисніть кнопку “Search”
      • Зверніть увагу на рядок “Minimum System Firmware”. Якщо її значення вище, ніж 13.00, то гра не запуститься на вашій приставці після встановлення патча!
      • Якщо на сайті немає патча для вашої гри, що підходить для прошивки 13.00 або нижче, пошукайте його в цьому архіві. Обов’язково перевірте, чи сумісний завантажений патч із прошивкою 13.00 або нижче. Зробити це можна методом із частини I через orbis-pub-chk.exe.
    3. Скачайте файл формату .pkg, що знаходиться в розділі Piece 0
    4. Якщо крім Piece 0 є ще й Piece 1 і так далі, то вам потрібно буде завантажити і їх, а потім об’єднати всі завантажені патчі в один
      1. Розархівуйте утиліту pkg-merge до завантажених частин патча
      2. Створіть папку “pkg” і перенесіть до неї всі патчі
      3. Запустіть pkg-merge.bat
      4. Дочекайтеся завершення процесу. На виході ви отримаєте великий файл патча, який і потрібно буде встановити
    5. Встановіть .pkg-файл патча таким самим способом, як і гру
