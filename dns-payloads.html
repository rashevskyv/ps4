    Описание пейлоадов, доступных через DNS

    Описание актуально на 26.06.2020 года.

    5.05
    • Mira - MiraCFW
    • Infinix - позволяет передавать игры через Store remote tool
    • Linux - необходим для запуска Linux
    6.72
    • App2USB - позволяет копировать игры из внутреннего HDD на внешний USB-носитель для того, чтобы играть потом используя внешний HDD и экономить место на внутреннем. Как пользоваться рассказано здесь
    • Backup - делает резервные копии сохранений и базы данных.
    • Bin Loader - слушает порт 9020 и ожидает загрузку пейлоада
    • Disable Updates - отключает возможность получать обновления. Удаляет существующее загруженное обновление и блокирует последующую загрузку
    • Dumper - дампит игры
    • Enable Updates - включает возможность получать обновления. Используйте только если хотите поставить прошивку выше, установленной. Помните, что в таком случае взлом будет невозможен!
    • FTP - FTP-сервер
    • History Blocker - Запрещает браузеру приставки запоминать последнюю открытую страницу. Браузер запускается со стартовой страницы. Для отключения, запустите ещё раз
    • Kernel Dumper - дампит ядро прошивки на USB-накопитель
    • Mira - MiraCFW, содержит HEN. Именно её нужно запускать для взлома
    • Mira (no HB) - Альтернативная MiraCFW. Так же содержит HEN и только его. Запускайте его при ошибке CE-34878-0
    • Restore - восстанавливает данные, сохранённые пейлоадом Backup. Подробнее делает резервные копии
    PS4
    • App2USB - позволяет копировать игры из внутреннего HDD на внешний USB-носитель для того, чтобы играть потом используя внешний HDD и экономить место на внутреннем. Как пользоваться рассказано здесь
    • Backup - делает резервные копии сохранений и базы данных.
    • Bin Loader - слушает порт 9020 и ожидает загрузку пейлоада
    • Disable Updates - отключает возможность получать обновления. Удаляет существующее загруженное обновление и блокирует последующую загрузку
    • Dumper - дампит игры
    • Enable Browser -
    • Enable Updates - включает возможность получать обновления. Используйте только если хотите поставить прошивку выше, установленной. Помните, что в таком случае взлом будет невозможен!
    • FTP - FTP-сервер
    • Fan Threshold - пейлоад для управлением максимальной скорости вращения турбины в приставке
    • HEN - Homebrew Enabler. Именно его нужно запускать для взлома. Содержит в себе спуфер прошивки
    • History Blocker - Запрещает браузеру приставки запоминать последнюю открытую страницу. Браузер запускается со стартовой страницы. Для отключения, запустите ещё раз
    • Kernel Clock - не знаю что делает
    • Kernel Dumper - дампит ядро прошивки на USB-накопитель
    • REN - Remote Play Enabler. Нужно запустить для активации возможности добавить Remote Play-устройство в консоль. Заменяет телодвижения через IDU-mode или Orbis
    • RIF Renamer - переименовывает ‘фейковые’ RIF-файлы в ‘свободные’ RIF-файлы для лучшей совместимости с HEN. Используйте если вы устанавливали PKG через Mira+HEN, а сейчас используете просто HEN
    • Restore - восстанавливает данные, сохранённые пейлоадом Backup. Подробнее делает резервные копии
