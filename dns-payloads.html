    6.72 / 7.05 / 7.55

    6.72 / 7.05 / 7.55

    Містить у собі пейлоади для 6.72 / 7.05 / 7.55

    • App2USB - дозволяє копіювати ігри з внутрішнього HDD на зовнішній USB-носій для того, щоб грати потім використовуючи зовнішній HDD та економити місце на внутрішньому. Як користуватися розказано тут
    • Backup - робить резервні копії збережень та бази даних.
    • Bin Loader - слухає порт 9020 та очікує завантаження пейлоада
    • Disable Updates - відключає можливість отримувати оновлення. Видаляє існуюче завантажене оновлення та блокує подальше завантаження
    • Dumper - дампить ігри
    • Enable Updates - вмикає можливість отримувати оновлення. Використовуйте тільки якщо хочете поставити прошивку вище, ніж встановлена. Пам’ятайте, що в такому разі злам буде неможливий!
    • FTP - FTP-сервер
    • History Blocker - Забороняє браузеру приставки запам’ятовувати останню відкриту сторінку. Браузер запускається зі стартової сторінки. Для відключення запустіть ще раз
    • Kernel Dumper - дампить ядро прошивки на USB-накопичувач
    • Mira - MiraCFW, містить HEN. Саме її потрібно запускати для зламу
    • Mira (Unofficial) - Альтернативна MiraCFW.

    • Restore - відновлює дані, збережені пейлоадом Backup. Детальніше робить резервні копії

    PS4

    Містить у собі пейлоади для 5.05

    • App2USB - дозволяє копіювати ігри з внутрішнього HDD на зовнішній USB-носій для того, щоб грати потім використовуючи зовнішній HDD та економити місце на внутрішньому. Як користуватися розказано тут
    • Backup - робить резервні копії збережень та бази даних.
    • Bin Loader - слухає порт 9020 та очікує завантаження пейлоада
    • Disable Updates - відключає можливість отримувати оновлення. Видаляє існуюче завантажене оновлення та блокує подальше завантаження
    • Dumper - дампить ігри
    • Enable Browser -
    • Enable Updates - вмикає можливість отримувати оновлення. Використовуйте тільки якщо хочете поставити прошивку вище, ніж встановлена. Пам’ятайте, що в такому разі злам буде неможливий!
    • FTP - FTP-сервер
    • Fan Threshold - пейлоад для керування максимальною швидкістю обертання турбіни у приставці
    • HEN - Homebrew Enabler. Саме його потрібно запускати для зламу. Містить у собі спуфер прошивки
    • History Blocker - Забороняє браузеру приставки запам’ятовувати останню відкриту сторінку. Браузер запускається зі стартової сторінки. Для відключення запустите ще раз
    • Kernel Clock - не знаю що робить
    • Kernel Dumper - дампить ядро прошивки на USB-накопичувач
    • REN - Remote Play Enabler. Потрібно запустити для активації можливості додати Remote Play-пристрій у консоль. Замінює маніпуляції через IDU-mode або Orbis
    • RIF Renamer - перейменовує ‘фейкові’ RIF-файли у ‘вільні’ RIF-файли для кращої сумісності з HEN. Використовуйте, якщо ви встановлювали PKG через Mira+HEN, а зараз використовуєте просто HEN

    • Restore - відновлює дані, збережені пейлоадом Backup. Детальніше робить резервні копії
