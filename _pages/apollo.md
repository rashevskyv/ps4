**Apollo Save Tool** — це найзручніший інструмент для керування збереженими даними (сейвами) та трофеями безпосередньо на PS4. Він позбавляє потреби використовувати ПК або платні сервіси на кшталт Save Wizard для більшості завдань.

**Важливо:** Для повноцінної роботи (експорту та перепідпису сейвів від інших користувачів) ваш профіль має бути **активований** (реальним PSN ID або фейковою активацією).
{: .notice--warning}

---

## Основні поняття

*   **Title ID:** Код гри (наприклад, `CUSA00744`). Різні регіони однієї гри мають різні Title ID.
*   **User ID:** Внутрішній ідентифікатор локального профілю (наприклад, `11cd8de`).
*   **Account ID:** 16-бітний HEX-код, прив'язаний до PSN. Саме до нього "прив'язуються" сейви.

---

## Індикатори сейвів у меню

Поруч із назвою збереження ви побачите іконки:
1.  **Замок (справа):** Сейв зашифрований. Якщо замка немає — сейв дешифрований.
2.  **Закладка (справа):** Account ID сейва збігається з ID поточного користувача.
3.  **Зірка (зліва):** Сейв вибрано (натисканням на тачпад) для масових дій.
4.  **Червоний знак оклику (!):** Помилка структури або сейв від вищої прошивки, для якої змінилися ключі.

---

## Можливості керування

### Робота з HDD (Внутрішні сейви)
*   **Apply Changes & Resign:** Застосувати чити та перепідписати сейв.
*   **Copy save game:** Копіювання дешифрованого сейва на USB за шляхом `/PS4/APOLLO/<USER-ID>/<TITLE-ID>`.
*   **Export save game to Zip:** Створення зашифрованого архіву на USB.
*   **Export/Import Keystone:** Резервне копіювання ключа Keystone (див. розділ "Виправлення").

### Робота з USB
*   **Resign:** Перепідпис сейвів під ваш поточний Account ID.
*   **Copy to HDD:** Встановлення сейва з флешки на консоль.

---

{% spoiler Масове керування (Bulk Management) %}

Ви можете вибирати кілька сейвів тачпадом або вибрати всі одразу в меню:
*   **HDD:** Копіювати всі вибрані сейви на USB у дешифрованому вигляді.
*   **USB:** Перепідписати (Resign) всі дешифровані сейви на флешці під ваш акаунт.

{% endspoiler %}

{% spoiler Спеціальні функції %}

### 1. Fake Activation (Фейкова активація)
Дозволяє перетворювати локальний офлайн-профіль на "активований":
1.  Перейдіть у "**User Tools**" -> "**Activate PS4 Accounts**".
2.  Виберіть профіль та введіть свій PSN Account ID (якщо хочете сумісність із хмарними сейвами) або дозвольте Apollo згенерувати випадковий.
3.  Натисніть "**Activate**" та перезавантажте систему.

### 2. Fake USB
Apollo може емулювати USB-накопичувач, використовуючи папку `/data/fakeusb` на внутрішньому HDD.
*   Це корисно, якщо у вас немає вільної флешки.
*   Увімкнути можна в налаштуваннях: "**USB Saves Source**".

### 3. Відбудова бази даних (App.db Management)
Якщо після збою або відбудови офіційної бази зникли іконки ігор/Homebrew:
1.  У "**User Tools**" виберіть "**App.db Database Management**".
2.  Натисніть "**Rebuild App.db Database**". Це поверне всі іконки на головний екран.
3.  Тут же можна зробити бекап всієї бази даних системи.

### 4. Виправлення пошкоджених сейвів (Keystone)
Якщо сейв має правильний ID, але гра каже "Пошкодено", проблема в Keystone (паролі розробника):
1.  Створіть новий збережений файл у цій же грі.
2.  В Apollo виберіть новий сейв і натисніть "**Export Keystone**".
3.  Виберіть "пошкоджений" сейв і натисніть "**Import Keystone**".

{% endspoiler %}

{% spoiler Конвертація між регіонами та версіями %}

Якщо ви хочете використати сейв від США версії (CUSA12345) на Європейській (CUSA54321):

**Спосіб 1 (Простий):**
1.  Зробіть новий сейв у цільовій грі.
2.  В Apollo виберіть його на HDD та натисніть "**Export decrypted save files**".
3.  Через **PS4Xplorer** перейдіть у `/data/apollo/<your-id>/` та перейменуйте папку старого Title ID на новий.
4.  В Apollo виберіть нову гру та натисніть "**Import decrypted save files**", потім "**Apply Changes & Resign**".

**Спосіб 2 (Для іншого регіону):**
1.  Дешифруйте обидва сейви (Source та Target) на USB через "**Copy save game**".
2.  На ПК замініть файли в папці Target файлами з Source (крім папки `sce_sys`).
3.  В Apollo скопіюйте Target сейв назад на HDD.

{% endspoiler %}

{% spoiler Мережеві функції та FTP %}

*   **Online Database:** Завантаження готових сейвів від спільноти прямо на HDD або USB.
*   **Власний FTP:** Ви можете налаштувати Apollo на роботу з вашим власним FTP-сервером (PC/NAS/Android) для бекапу сейвів по повітрю.
*   **Web Server:** Apollo запускає локальний сервер (`http://ip_ps4:8080`), де можна через браузер на ПК переглядати та завантажувати файли консолі.

{% endspoiler %}

## Корисні поради
*   **Трофеї:** Apollo може робити бекап трофеїв у дешифрованому вигляді, що дозволяє переносити їх між консолями.
*   **Чити:** Якщо гра підтримується, у меню сейва з'явиться пункт "**Cheats**". Виберіть потрібні, натисніть "**Apply changes**".
*   **Debug Log:** Якщо програма працює нестабільно, увімкніть логування в налаштуваннях (`/data/apollo/apollo.log`).
