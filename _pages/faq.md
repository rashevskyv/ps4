{% spoiler Чи можна знизити прошивку? %}
Ні, на даний момент даунгрейда не існує. І ні, ви не зможете встановити стару прошивку, просто помінявши жорсткий диск. 
{% endspoiler %}

{% spoiler Приставка завантажила оновлення, як видалити? %}

Просто запустіть GoldHEN. Блокувальник вбудований в експлойт.

{% endspoiler %}

{% spoiler Як дізнатися версію прошивки за серійним номером ревізії? %}
За допомогою цього [сайту](https://defaultdnb.github.io/PS4IDENT/index.html){:target="_blank"}
{% endspoiler %}

{% spoiler Це що, кожного разу потрібно запускти експлойт? %}
Так, при холодному ввімкненні приставки. Ні, якщо ви виводите приставку з режиму сну і експлойт був уже запущений, коли приставка заснула.
{% endspoiler %}

{% spoiler Чи зникнуть ігри, які я встановив на минулих прошивках при оновленні на останню підтримувану версію ПЗ? %}
Ні, звичайно. Навіть при оновленні через рекавері. Але я б все одно рекомендував зробити резервні копії збережень стандартними способами. Не знаєте як - подивіться в інструкції до приставки. 
{% endspoiler %}

{% spoiler Не можу знайти налаштування Debug Settings після запуску Mira %}
З якоїсь причини з Mira прибрали це налаштування. Використовуйте **Mira (Unofficial)**
{% endspoiler %}

{% spoiler Як видалити патч або DLC, залишивши саму гру? %}
Запускаєте злам, переходьте до налаштувань, там **Debug Settings** -> **Game** -> **Add Content Manager** -> **Entitlement Control**
Будьте обережні, не видаліть зайвого!
{% endspoiler %}

{% spoiler Як зайти в PSN? %}
Ніяк. Увійти до PSN можна тільки оновившись на останню офіційну прошивку!
{% endspoiler %}

{% spoiler Як закешувати експлойт заново? %}
Зайдіть у браузер та видаліть дані (у браузері натисніть (OPTIONS) -> **Налаштування** -> **Видалити дані веб-сайтів**), потім відкрийте посібник користувача та натисніть **cache**. Пам'ятайте, що у вас має бути [налаштований DNS](start-hen){:target="_blank"}!
{% endspoiler %}

{% spoiler Як запустити VR на зламаній приставці? %}
Якщо шолом попросить оновити прошивку – зробіть це. Підключіть та ввімкніть VR **перед** активацією зламу. 
{% endspoiler %}

{% spoiler Чи можна використовувати Remote Play? %}
[Так](remote-play){:target="_blank"} 
{% endspoiler %}

{% spoiler Як запустити гру від PS2 на PS4? %}
Дотримуйтесь [рекомендацій з цього відео](https://youtu.be/YKurUS2_wC0){:target="_blank"}
{% endspoiler %}

{% spoiler Чи працюватимуть збереження від ліцензійних ігор? %}
Ви зможете використовувати збереження на ліцензійних іграх і надалі. Проте на піратських копіях збереження, зроблені на ліцензії, працювати не будуть! Вони працюватимуть тільки якщо ви будете використовувати їх на ліцензійних копіях ігор! 
{% endspoiler %}

{% spoiler Чи потрібно оновлюватися до останньої версії? Мені і на 5.05 непогано %}
Ваше право не оновлюватися. Проте знайте, що на 9.00 експлойт працює так само добре, як і на 5.05 (однак пам'ятайте, що експлойт на 9.00 для запуску вимагає підготовлену спеціальним чином флешку) і не вимагає бекпортів
{% endspoiler %}

{% spoiler Як перенести ігри з внутрішнього HDD на зовнішній USB-носій? %}
Способом, [передбаченим інженерами Sony](https://blog.ru.playstation.com/2017/03/11/external-hdd-faq/){:target="_blank"}
{% endspoiler %}

{% spoiler У якому порядку встановлювати PKG? %}
Порядок встановлення гри, патчів та DLC може бути довільним, якщо додатково не сказано іншого. 
{% endspoiler %}

{% spoiler Чи потрібно перекачувати куплені ліцензійні ігри? %}
Не обов'язково. Використовуйте пейлоадер [**reactPSPlus**](https://github.com/Zer0xFF/reactPSPLUS/releases/latest){:target="_blank"} для активації цих ігор
{% endspoiler %}

{% spoiler Де брати патчі на ігри? %}
Ніде. Патч потрібно дампити так само, як і гру. Зазвичай у роздачі з грою йде патч, якщо такого там немає, то знайти його в більшості випадків не вийде. Ігри та патчі з сайтів типу [https://orbispatches.com/](https://orbispatches.com/){:target="_blank"} не підходять, оскільки такі ігри не є перепакованими для зламаної прошивки. 
{% endspoiler %}

{% spoiler Чи можу я прошити приставку з прошивкою відмінною від 9.00? %}
Так, існує метод запуску ігор для прошивки 1.76, 4.05, 4.55, 5.05, 6.72, 7.02, 7.55, але в цьому посібнику розглядається злам тільки для найактуальнішої прошивки, що зламується. Ви можете [підняти версію вашої прошивки](usb-update-900){:target="_blank"}, якщо вона у вас нижча, ніж 9.00, або залишитися на стабільній 6.72.
{% endspoiler %}

{% spoiler Чи вистачить флешки на XX Гб для встановлення ігор? %}
Розмір флешки не повинен бути меншим за розмір гри, що очевидно. Рекомендую носій не менше 64Гб або зовнішній HDD
{% endspoiler %}

{% spoiler Чи можна встановити гру, перекинувши її по FTP або будь-яким іншим методом? %}
Так. Якщо у вас немає накопичувача великого розміру або з якихось причин ви не можете скористатися способом встановлення ігор через нього, скористайтеся [**Встановленням по локальній мережі**](games){:target="_blank"}. 
{% endspoiler %}

{% spoiler Чи можна розбити pkg на частини, щоб влізла на флешку? %}
Ні.
{% endspoiler %}

{% spoiler Чи можна грати онлайн? %}
Ні. Навіть не намагайтеся увійти до PSN з активованим зламом! Це небезпечно і майже напевно призведе до бану
{% endspoiler %}

{% spoiler Чи можу я запустити ігри, які вимагають прошивку вище за 9.00? %}
Ні. І ніколи не зможете, якщо хтось не сдампить гру, використовуючи експлойт на прошивці вище і не модифікує її для запуску на вашій прошивці.
{% endspoiler %}

{% spoiler Як повернутися на офіційну прошивку? %}
Формально ви з неї нікуди і не йшли. Подробиці [тут](uninstall-hen){:target="_blank"}
{% endspoiler %}

{% spoiler Чи можна пропатчити гру до актуальної версії? %}
Якщо гра ліцензійна, то перед дампом її потрібно патчити таким чином - [Застосування ігрових патчів на ліцензійні копії ігор](game-patches){:target="_blank"}. Патчі ж для піратських копій шукайте там же, де ви берете дампи ігор. 
{% endspoiler %}

{% spoiler У мене немає можливості надати активне інтернет-з'єднання для консолі. Що мені робити? %}
Ви можете передати пейлоадер через мобільний телефон (див. нижче) або скористатися [USB WiFi-модулем ESP](https://4pda.to/forum/index.php?showtopic=885825&view=findpost&p=73705006){:target="_blank"}
{% endspoiler %}

{% spoiler Чи можна хостити експлойт на мобільному телефоні? %}
Так, для [Android](https://4pda.to/forum/index.php?showtopic=885825&view=findpost&p=70298081){:target="_blank"} та [iOS](https://www.psxhax.com/threads/how-to-host-ps4hen-for-airdisk-ios-iphone-guide-by-leslie84.4383/){:target="_blank"} є відповідне ПЗ
{% endspoiler %}
