    Vue After Free — це експлойт PlayStation 4 користувацького рівня (userland). Для злому приставки використовується офіційний застосунок Vue та вразливість, яка дозволяє завантажувати довільний код Kexploit. Це викликає спеціально прописану ланцюгову реакцію і призводить до злому приставки.

    • Для використання цього методу приставка має бути на прошивці 7.00-13.00
    • У разі невдалого злому завжди перезапускайте PS4. Повторний запуск Kernel експлойта без перезавантаження погіршить роботу експлойта, або спровокує Kernel Panic (“KP”).
    • Без серйозної необхідності встановлення апдейтів бекапу не рекомендується! Якщо все ж вирішили зробити апдейт через відновлення бекапу, рекомендується витягнути свої ігрові збереження, щоб після відновлення імпортувати їх назад.
    • Вкрай рекомендується періодично робити власний бекап системи за допомогою самої PS4, включаючи в нього Vue After Free Jailbreak та ваші ігрові збереження. Детальніше про бекап і збереження.

    Повна інформація про методи

    • Метод - бекап: готова резервна копія системи PS4 (complete restore image). Це повноцінний системний бекап, створений на робочій консолі з уже встановленим та налаштованим Vue After Free. Включає в себе повний образ системи з налаштуваннями, офіційним застосунком Vue, файлом download0.dat та всіма необхідними директоріями (cache, splash-ресурси тощо). Дозволяє швидко розгорнути робочу конфігурацію без ручного налаштування кожного кроку. Призначений виключно для незламаних консолей.
    • Метод - download0: метод для прошитої консолі. Включає в себе всі необхідні структури каталогів (cache/, splash_screen/ і т.д.). download0.dat, що дозволяє розгорнути метод Vue After Free.

    Встановлення DNS в консоль

    Встановлення DNS в консоль

    1. Увімкніть PS4
    2. Перейдіть до налаштувань, виберіть “Мережа”
    3. Поставте галочку навпроти “Підключити до Інтернету”
    4. Перейдіть до “Встановити з’єднання з Інтернетом”
    5. Виберіть “Використовувати Wi-Fi” або “Використовувати LAN” залежно від типу вашого підключення — бездротового чи дротового відповідно
    6. Виберіть “Спеціальний”, виберіть ваше підключення Wi-Fi, якщо вибрано підключення через Wi-Fi
    7. У меню “Налаштування IP-адреси” виберіть “Автоматично”
    8. У меню “Ім’я хоста DHCP” виберіть “Не вказувати”
    9. У меню “Налаштування DNS” виберіть “Вручну”
    10. У полі “Основний DNS” введіть 62.210.38.117
    11. У полі “Додатковий DNS” залиште 0.0.0.0
    12. Натисніть “Далі”
    13. У меню “Налаштування MTU” виберіть “Автоматично”
    14. У меню “Проксі-сервер” виберіть “Не використовувати”
    15. Натисніть “Перевірити з’єднання з інтернетом” для завершення налаштування
      • Якщо приставка пише, що вдалося отримати IP-адресу, але не вдалося підключитися до інтернету — це нормально, це значить що з’єднання з інтернетом є, а ось з серверами соні — ні
    16. Після завершення перевірок натисніть кнопку (PS), щоб повернутися на початковий екран

    Приставка вже взламана

    Ви можете встановити цей додаток на вже взломану приставку щоб мати простіший спосіб запуску експлойта, або щоб мати можливість запускати його без інтернету

    Оберіть один з наступних методів:

    Спрощене встановлення Vue Initial Files

    Спрощене встановлення Vue Initial Files

    Вам знадобиться будь-який USB-накопичувач (флешка).

    1. Переконайтеся, що ви налаштували DNS в налаштуваннях мережі (див. спойлер вище).
    2. Відформатуйте USB-накопичувач на ПК у файловій системі exFAT.
    3. Завантажте і розпакуйте в корінь флешки: Vue_base_files.zip.
      • На флешці має стати 01_Vue_base_v1.01.pkg, 02_Vue_update_v1.24.pkg, 03_Vue_Initial_Files_2.0.pkg та папка PS4
    4. Завантажте останню версію Apollo Save Tool (файл IV0000-APOL00004_00-APOLLO0000000PS4.pkg).
    5. Завантажте останню версію GoldHEN (файл GoldHEN_v2.x.7z під спойлером Assets), покладіть goldhen.bin в корінь вашої картки пам’яті з іменем payload.bin
    6. Встановіть через GoldHEN -> Debug Settings -> Package Installer (меню знаходиться над списком ігор або на самому початку Налаштувань) всі .pkg файли суворо по черзі:
      1. 01_Vue_base_v1.01.pkg
      2. 02_Vue_update_v1.24.pkg
      3. 03_Vue_Initial_Files_2.0.pkg
      4. IV0000-APOL00004_00-APOLLO0000000PS4.pkg
    7. Запустіть встановлений застосунок Vue Initial Files, допоміжні файли будуть скопійовані автоматично.
      • Можете видалити цей застосунок після того як він закінчіть копіювання
    8. Переконайтеся, що ваш PS4 акаунт активований (у PSN або фейково через Apollo Save Tool > User Tools > Activate PS4 Accounts, не активований акаунт матиме id ‘0000000000000000’).
    9. Запустіть застосунок Apollo Save Tool, перейдіть до розділу USB Saves.
    10. Виберіть у списку збереження PS Vue (CUSA00960) і скористайтеся опцією Copy to HDD (або Copy save game to HDD), щоб перенести його на консоль. (Це збереження не потребує перепідпису, оскільки воно генерується безпосередньо на вашій консолі).
    11. Перезавантажте консоль і запустіть застосунок Vue не запускаючи попередньо взлом іншим способом
    12. Під час запуску виберіть режим:
      • Lite install (Спрощений режим без зайвих налаштувань).
    13. Натисніть кнопку Jailbreak. Дочекайтеся сповіщення “GoldHEN by Sistro”.
    14. Зробіть резервну копію збереження Vue на флешку про всяк випадок. Для цього вставте USB-накопичувач у форматі exFAT і перейдіть у Налаштування -> Керування збереженими даними додатків -> Збережені дані в пам’яті системи -> Копіювання на накопичувач USB.
      • Детальніше про бекапи

    Відновлення бекапу (Із втратою даних)

    Відновлення бекапу (Із втратою даних)

    Зробите резервну копію важливих сейвів та акаунтів, інакше всі ваші дані будуть видалені та втрачені. Для цього вставте USB-накопичувач у форматі exFAT і перейдіть у Налаштування -> Керування збереженими даними додатків -> Збережені дані в пам’яті системи -> Копіювання на накопичувач USB. * Детальніше про бекапи

    Увага: перед виконанням процедури настійно рекомендується попередньо виконати повне скидання налаштувань консолі до заводських значень. Всі ваші дані на консолі будуть видалені! Бекап зроблений на прошивці 07.00, використовувати його можна на прошивках від 07.00 до 13.00.

    1. Переконайтеся, що ви налаштували DNS в налаштуваннях мережі (див. спойлер вище).
    2. Завантажте VueSystemBackup.7z
    3. Відформатуйте USB-накопичувач на ПК у форматі exFAT.
    4. Розпакуйте вміст бекапу на USB-накопичувач. Має вийти наступна структура: /PS4/EXPORT/BACKUP/2026.../archive.dat
    5. Підключіть USB-накопичувач до PS4 і відновіть бекап на консолі: перейдіть у Налаштування -> Система -> Створення і відновлення резервних копій -> Відновити PS4.
    6. Чекайте закінчення відновлення. Коли воно завершиться, приставка автоматично перезавантажиться.
    7. Переконайтеся, що ви налаштували локальну мережу, вказавши основний DNS: 62.210.38.117.
    8. Завантажте останню версію GoldHEN (файл GoldHEN_v2.x.7z під спойлером Assets), покладіть goldhen.bin в корінь вашої картки пам’яті з іменем payload.bin
    9. Запустіть застосунок PlayStation Vue (без попередньої активації іншого злому).
      • Якщо застосунок Vue запропонує підключитися до “PSN”, просто натисніть кнопку (Х) або (О).
    10. Під час запуску виберіть режим:
      • Lite install (Спрощений режим без зайвих налаштувань).
    11. Натисніть кнопку Jailbreak. Після завершення ви побачите повідомлення “GoldHEN by Sistro”.
    12. Важливо: Зробіть резервну копію щойно встановленого збереження програми Vue на вашу флешку про всяк випадок. Для цього вставте USB-накопичувач у форматі exFAT і перейдіть у Налаштування -> Керування збереженими даними додатків -> Збережені дані в пам’яті системи -> Копіювання на накопичувач USB. Виберіть застосунок Vue та скопіюйте його.
    Відновлення бекапу (Із втратою даних)

    Для оновлення версії GoldHEN, скачайте останній реліз GoldHEN (файл GoldHEN_v2.x.7z під спойлером Assets), покладіть goldhen.bin в корінь вашої картки пам’яті з іменем payload.bin та запустіть взлом. GoldHEN у вигляді payload.bin буде скопійований всередину приставки.

    Для оновлення плейграунда всередині Vue на інший, виконайте копіювання download0

    Якщо взлом активовано правильно, на головному екрані у верхній смужці має з’явитися значок GoldHEN, що вказує на те, що злом активовано успішно. Також меню GoldHEN можна знайти над іграми (поруч із піктограмою «Що нового») або у самому верху Налаштувань консолі.

    На цьому етапі ми вже активували GoldHEN і можна повноцінно користуватися консоллю.
