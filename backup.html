    Резервне копіювання та відновлення відбувається відразу для всіх профілів!

    Відео-інструкція
    Відео-інструкція

    Цей посібник охоплює як офіційні методи для будь-якої PS4, так і специфічні способи для власників консолей із прошивкою HEN. Нижче наведено максимально детальне пояснення логіки роботи системи «для новачків».

    Логіка роботи бекапів на PS4

    Коли ви вперше вмикаєте PS4 і створюєте користувача (наприклад, User1), система присвоює йому унікальний ідентифікатор — випадкову комбінацію букв та цифр.

    Ви не побачите цей ID у меню, але його можна дізнатися через FTP за адресою: UserData/user/home/XXXXXXXX/ (ці 8 символів і є вашим кодом).

    Важливо розуміти:

    • Прив’язка сейвів: Всі ваші збереження (прогрес ігор) намертво пришиті саме до цього профілю та його ID. На іншій консолі вони не працюватимуть, бо там у профілів інші ID.
    • Проблема ініціалізації: Навіть у межах однієї консолі, якщо ви зробите ініціалізацію (повний скид), а потім створите користувача з таким самим ім’ям — система згенерує НОВИЙ ID. Старі сейви перетворяться на «гарбузи» (стануть недійсними).
    • Неможливість підробки: Підробити старий ID неможливо, він зашитий занадто глибоко. Ручне перенесення файлів у папку з новим ID також не спрацює.

    Що ми робимо?

    Для власників зламаних консолей вихід один: робити резервну копію не лише самих сейвів, а й самого профілю. Ми робимо «зліпок» користувача разом із його унікальним ID, щоб після формування не перестворювати його, а відновити старий.

    Частина 1. Резервне копіювання профілю (Офіційний метод)

    Це робиться за допомогою стандартних засобів системи. Рекомендується робити бекап пустого профілю (без встановлених ігор):

    1. Компактність: Бекап з іграми може займати сотні гігабайт, а пустий профіль — близько 50 МБ.
    2. Швидкість: Він створюється і відновлюється миттєво.
    3. Стабільність: Чистий профіль зводить до мінімуму ризик «крашів» та помилок системи.

    Зробивши такий бекап один раз, ви зможете відновлювати на цей профіль будь-які сейви скільки завгодно.

    Як зробити бекап профілю:

    1. Підключіть USB-накопичувач (FAT32/exFAT).
    2. Перейдіть у “Налаштування” -> “Система” -> “Створення резервної копії та відновлення” -> “Створити резервну копію даних PS4”.
    3. Порада: Вимкніть пункт “Програми”, щоб зберегти лише профіль та налаштування.
    4. Чекайте завершення. На флешці з’явиться папка PS4. Збережіть її на комп’ютері та бережіть як зіницю ока.

    Частина 2. Резервне копіювання сейвів та трофеїв (для HEN)

    Коли профіль вже «забекаплений», вам залишається лише періодично зберігати прогрес ваших ігор.

    2.1. Метод «Олдскул» — ручне копіювання через FTP (Найнадійніший)

    Це дозволяє копіювати конкретні ігри поштучно.

    1. Підключіть консоль до мережі та активуйте пейлоад FTP (наприклад, через GoldHEN або хост з експлойтом).
    2. Через ПК (FileZilla) перейдіть за шляхом:
      • Сейви: user/home/XXXXXXXX/savedata/CUSAXXXXX
        • XXXXXXXX — ваш унікальний ID.
        • CUSAXXXXX — код гри (можна дізнатися в Google або базах ігор).
      • Трофеї: user/home/XXXXXXXX/trophy (копіюйте, якщо хочете зберегти «кубки» та «платини»).
    3. Просто скопіюйте ці папки на комп’ютер.

    2.2. Через пейлоад (Backup DB)

    1. Активуйте пейлоад BackupDB на хості з експлойтами.
    2. Важливо: Обов’язково підключіть консоль до інтернету перед запуском (з налаштованими DNS), інакше пейлоад може зависнути.
    3. Файли з’являться на USB у папках DB_Backup та UserData.

    2.3. Через Apollo Save Tool (Найзручніший)

    Це найбільш сучасний та функціональний спосіб, який працює без ПК.

    1. Запустіть Apollo Save Tool на консолі.
    2. Перейдіть у “User Saves” -> “HDD”.
    3. Для одного сейва: Оберіть гру зі списку та натисніть “Copy save game” (скопіює розшифровані файли) або “Export save game to Zip” (створить архів).
    4. Для всіх сейвів (Масове копіювання):
      • Натисніть на тачпад контролера, щоб увімкнути режим вибору (з’являться зірочки).
      • Виберіть потрібні ігри або натисніть “Options” -> “Select all”.
      • Виберіть пункт “Copy all selected to USB”.
    5. Сейви з’являться на флешці за шляхом /PS4/APOLLO/.

    Частина 3. Відновлення даних

    3.1. Якщо пошкоджено лише сейв окремої гри

    Зазвичай профіль не страждає. Достатньо повернути файли гри на місце:

    1. Підключіться по FTP.
    2. Видаліть папку гри (CUSAXXXXX) на консолі, де лежать пошкоджені дані.
    3. Завантажте справну копію з вашого бекапу на ПК.
    4. Перезавантажте приставку.

    3.2. Відновлення через Apollo Save Tool

    Якщо у вас є сейви на флешці (у папці /PS4/APOLLO/):

    1. Запустіть Apollo Save Tool.
    2. Перейдіть у “User Saves” -> “USB”.
    3. Виберіть потрібну гру.
    4. Натисніть “Copy to HDD”.
    5. Якщо сейв був від іншого акаунта, натисніть “Apply Changes & Resign”, щоб він запрацював на вашому профілі.

    3.3. Якщо «все дуже погано» (Відновлення профілю)

    Це потрібно після ініціалізації консолі (повного скидання до заводських). Увага: Відновлення профілю зітре всі поточні дані на консолі! Перед початком переконайтеся, що ви зробили окремі копії нових сейвів через FTP або Apollo.

    1. Якщо була ініціалізація — створіть будь-якого користувача (ім’я не має значення).
    2. Перейдіть у “Налаштування” -> “Система” -> “Створення резервної копії та відновлення” -> “Відновити дані PS4”.
    3. Оберіть ваш бекап із USB.
    4. Після перезавантаження ваш старий профіль із правильним ID повернеться. Тепер ви можете завантажити в нього свої сейви через FTP, пейлоад або Apollo.

    Що таке ініціалізація?

    Це повне форматування жорсткого диска. Всі ігри, сейви та профілі будуть видалені. Це «крайній метод», який використовується лише для вирішення важких системних глюків. Не робіть її без гострої потреби.

    Додаткові інструменти

    Для більш гнучкої роботи з окремими сейвами (перепідпис під інший акаунт, чити, виправлення помилок) використовуйте Apollo Save Tool.
