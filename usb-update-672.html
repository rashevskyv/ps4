    Підготовчі роботи

    1. Відформатуйте USB-флешку в exFAT
    2. У корені USB-флешки створіть папку PS4, а в ній папку UPDATE
    3. Завантажте звичайний файл оновлення для простого підняття версії прошивки, або файл оновлення для рекавері, якщо ви хочете змінити жорсткий диск на приставці, або у вас вже завантажилося оновлення на останню версію ПЗ, що не піддається зламу
      • [Звичайний файл системного оновлення]https://archive.org/download/PS4-System-Firmwares/Firmware%206.72/PS4UPDATE.PUP для Playstation 4 версії 6.72
      • [Версія для рекавері]https://archive.org/download/PS4-Recovery-Firmwares/Firmware%206.71/PS4UPDATE.PUP для Playstation 4 версії 6.72
    4. Перенесіть файл оновлення в папку PS4/UPDATE і перейменуйте його в PS4UPDATE.PUP

    Звичайне оновлення через USB

    Звичайне оновлення через USB

    1. Вставте USB-накопичувач з файлами прошивки в USB-порт приставки
    2. Відкрийте Налаштування, перейдіть до “Оновлення системного програмного забезпечення”

    3. Приставка напише, що знайдено оновлення і вкаже його номер

      УВАЖНО ПЕРЕКОНАЙТЕСЯ, ЩО ОНОВЛЮЄТЕСЯ НА ВЕРСИЮ 6.72

    4. Завершіть встановлення оновлення, дотримуючись інструкцій на екрані
      • Якщо виникли проблеми, спробуйте прошити через рекавері

    Оновлення через рекавері

    Відео
    Відео
    1. Вимкніть систему PS4, натиснувши кнопку живлення на передній панелі. Індикатор живлення кілька разів блимне і згасне
    2. Вимкнувши систему PS4, знову натисніть і утримуйте кнопку живлення. Відпустіть її, почувши другий звуковий сигнал: один звуковий сигнал пролунає при натисканні на кнопку живлення, а другий — ще через кілька секунд
    3. Підключіть контролер DUALSHOCK 4 за допомогою кабелю USB і натисніть кнопку PS на контролері
    4. Виберіть пункт “Оновити системне програмне забезпечення”
    5. Виберіть пункт “Оновити з накопичувача USB” і натисніть ОК

    6. Приставка напише, що знайдено оновлення і вкаже його номер

      УВАЖНО ПЕРЕКОНАЙТЕСЯ, ЩО ОНОВЛЮЄТЕСЯ НА ВЕРСИЮ 6.72

    7. Завершіть встановлення оновлення, дотримуючись інструкцій на екрані
    Оновлення через рекавері

    Якщо виникає помилка на прошитій приставці (наприклад CE-30778-5 або SU-30634-6), насамперед переконайтеся, що файл оновлення названий PS4UPDATE.PUP. Якщо це не допомогло — видаліть вміст папки update на HDD вашої приставки, або скористайтеся пейлоадером update-enabler (є за замовчуванням у складі ps4-exploit-host)

    Якщо виникає помилка на не прошитій приставке, насамперед переконайтеся, що файл оновлення названий PS4UPDATE.PUP, спробуйте прошити з іншим USB-накопичувачем, спробуйте перекачати файл оновлення, спробуйте зробити повну ініціалізацію приставки. Якщо у вас проблеми з приводом, то скористайтеся цим відео

    Для заміни жорсткого диска використовуйте системне оновлення для рекавері. Вставте новий диск і встановіть це оновлення. Приставка сама підготує диск до роботи
